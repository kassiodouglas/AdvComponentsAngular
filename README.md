# @kassdoug

Arquivos de desenvolvimento dos componentes Tableadv e Sidebaradv.

## Instalação
Para instalar rode:
```
npm i @kassdoug/tableadv
npm i @kassdoug/sidebaradv
```

<br><br>

## Dev
Iniciar o app de testes:
```
npm run dev:app
```

Iniciar a lib e fazer build de testes:
```
npm run dev:table
npm run dev:sidebar
```

Para publicar
```
npm run pub:table
npm run pub:sidebar
```
